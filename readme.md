# 🎮 Launcher

A simple, fast, and user‑friendly **web game launcher** made for GitHub Pages.

## ✅ Features
- Automatically loads games
- Click icons to play instantly
- Icons auto‑resize to the same size
- Clean and modern design
- Easy to expand

## 📂 Adding New Games
1. Add a game folder in `/games`
2. Add an icon to `/thumb`
3. Add one entry in `games.json`
